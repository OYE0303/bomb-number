rule

  1. There is a bomb number in each game
  2. Any player is only allowed to guess the number in valid range
  3. You have to guess the number in 30 second, or it will automatically choose a random number for you
  4. Game is over if you guess the bomb number
  5. You can use different tools to avoiding guessing the number
  6. Every other players(computer) will have three tool counts for every level
  7. If one of other players guesses the bomb number
    1. Other remaining players can have one more tool to use
    2. A new bomb number is re-created
    3. A new valid number range is re-set
  8. You can only win the game if all other players lose the game
